# 非默 The Unmuted 🛡️

> *让真相不可磨灭，让受害者不再孤身一人。*
> *Make truth indelible; ensure no survivor stands alone.*

基于 Solana 构建的 Web3 匿名证据库与女性互助系统。  
Web3-powered anonymous evidence vault and women's mutual-aid system, built on **Solana**.

---

## ✨ Features

### 🌟 3.1 Secure Evidence Vault
- **AES-256-GCM** local encryption — keys never leave your device
- **Arweave** permanent decentralized storage for encrypted evidence
- **Solana Memo Program** on-chain timestamp anchoring
- One-tap capture: photo, video, and audio
- Downloadable decryption key bundle
- Phantom wallet support

### 🌟 3.2 Anonymous Community
- ZKP identity verification
- Anonymous geo-fenced danger alerts
- P2P encrypted support chat

### 🌟 3.3 DAO Support
- Legal and mental-health support governance
- Community voting and proposal flow

## 🛠️ Tech Stack

| Layer | Tech |
|---|---|
| Frontend | React 18 + TypeScript + Vite + Tailwind CSS |
| Mobile | PWA |
| Wallet | Phantom |
| Chain | Solana |
| Storage | Arweave |
| Encryption | Web Crypto API — AES-256-GCM |

## 🚀 Getting Started
